## 🔨 最简单的运行方法

### 方法一：修改现有 cpp-empty-test（最快）

如果你已经构建过 cocos2d-x 的测试项目，可以直接把我们的代码替换过去：

1. 将以下文件复制到 `tests\cpp-empty-test\Classes\`：
   - `Enemy.h/cpp`
   - `Tower.h/cpp`
   - `TestScene.h/cpp`

2. 修改 `tests\cpp-empty-test\Classes\AppDelegate.cpp`
   - 替换 `#include "HelloWorldScene.h"` 为 `#include "TestScene.h"`
   - 替换 `HelloWorld::scene()` 为 `TestScene::scene()`

3. 重新运行 cpp-empty-test

### 方法二：使用 CMake 完整构建（推荐）

#### 第一步：构建 Cocos2d-x 引擎（如果还没构建过）

```bash
cd e:\ProjCore\游戏\cocos2d-x-4.0
mkdir build
cd build
cmake .. -G "Visual Studio 17 2022" -A Win32
```

然后用 Visual Studio 打开 `build\Cocos2d-x.sln`，选择 Release 配置，构建解决方案。

#### 第二步：构建塔防游戏

```bash
cd e:\ProjCore\游戏\cocos2d-x-4.0\TowerDefense
build.bat
```

然后用 Visual Studio 打开 `build\TowerDefense.sln` 并运行。

## 🔄 想玩完整游戏？

修改 `AppDelegate.cpp` 中的这两行：

```cpp
// 把这两行：
#include "TestScene.h"
auto scene = TestScene::scene();

// 改成这两行：
#include "GameScene.h"
auto scene = GameScene::scene();
```
